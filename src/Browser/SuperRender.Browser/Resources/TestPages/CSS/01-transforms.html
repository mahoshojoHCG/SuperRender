CSS Test 01: Transforms

1a: translate(30px, 20px)
T
Box should be shifted 30px right and 20px down from normal position.
1b: rotate(45deg)
R
Box should be rotated 45 degrees clockwise around its center.
1c: scale(1.5)
S

1d: skewX(20deg)
K
Box should be skewed horizontally by 20 degrees.
1e: Multiple transforms
M
Box should be translated, rotated 30deg, and scaled to 80%.
1f: transform-origin: top left
O
