Test 2a: overflow: hidden with fixed height (80px)
This container has overflow:hidden and a fixed height of 80px. This text is intentionally long so that it overflows the container. You should NOT see this sentence because the container clips content. This line is definitely beyond the 80px height limit. And this line too. All of this should be invisible if overflow:hidden works.
EXPECT: Text is clipped at the bottom of the red-bordered box. No content visible below the 80px boundary.
Test 2b: overflow: hidden + white-space: nowrap + text-overflow: ellipsis (width: 250px)
This is a very long line of text that should be truncated with an ellipsis character at the end because the container is only 250px wide and white-space is nowrap.
EXPECT: Single line of text truncated with "..." at the right edge of the blue-bordered box.
Test 2c: overflow: visible (default) with fixed height (60px)
This container has overflow:visible and a fixed height of 60px. The text content exceeds the height. This text should bleed out below the green border. You should be able to read this even though it is outside the container.
EXPECT: Text overflows past the green border and is visible below the box. Content bleeds out.
