Test 12a: Opening DevTools

Open the Developer Tools using any of these methods:

  • Press F12
  • Press Cmd+Shift+I (macOS) or Ctrl+Shift+I
  • Right-click the page and select "Developer Tools" from the context menu
EXPECT: A separate DevTools window opens. It should have a console area showing the log messages from this page's script.
Test 12b: Console Output

After opening DevTools, check that the console displays:

  1. A LOG message (normal text)
  2. A WARN message (may have yellow/warning styling)
  3. An ERROR message (may have red/error styling)
  4. An object: { name: "test", value: 42 }
  5. An array: [1, 2, 3, 4, 5]
EXPECT: All 5 console outputs are visible in the DevTools console. Different log levels may have different visual styling.
Test 12c: Console Input (JS Execution)

In the DevTools console input area, try typing and executing these commands:

  1. 1 + 1 -- should output 2
  2. document.title -- should output "Test 12 - Developer Tools"
  3. document.body.childNodes.length -- should output a number
  4. var x = 42; x * 2 -- should output 84
EXPECT: Each expression is evaluated and the result is displayed in the console.
Test 12d: Console DOM Manipulation

Try executing DOM manipulation in the DevTools console:

  1. document.getElementById("target").textContent = "Modified by DevTools!"

This text should be changed by the DevTools command above.

EXPECT: After running the command, the text above changes to "Modified by DevTools!" and the page re-renders.
Test 12e: Toggle DevTools

Press F12 again (or Cmd+Shift+I) to close the DevTools window. Press it once more to reopen.

EXPECT: DevTools window toggles open/closed. Console history may be preserved across toggles.
