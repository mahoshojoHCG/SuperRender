Test 11a: Address Bar Context Menu

Steps:

  1. Click on the address bar at the top of the browser window
  2. Type some text into the address bar
  3. Right-click inside the address bar

Expected context menu items:

  • Cut - cuts selected text from address bar
  • Copy - copies selected text to clipboard
  • Paste - pastes from clipboard into address bar
  • Select All - selects all text in address bar
EXPECT: A context menu appears near the mouse cursor with the items listed above. Each item is clickable and performs its action.
Test 11b: Content Area Context Menu (no selection)

Steps:

  1. Make sure no text is selected on this page
  2. Right-click anywhere on this page content

Expected context menu items:

  • Select All - selects all text on the page
  • View Source - shows the page HTML source
  • Developer Tools - opens the DevTools window
EXPECT: Context menu appears without a "Copy" option (nothing is selected). "Select All", "View Source", and "Developer Tools" are present.
Test 11c: Content Area Context Menu (with selection)

Steps:

  1. Select some text on this page by clicking and dragging
  2. Right-click on the selected text

Expected context menu items:

  • Copy - copies the selected text to the clipboard
  • Select All - selects all text on the page
  • View Source - shows the page HTML source
  • Developer Tools - opens the DevTools window

Here is some text you can select for testing: The quick brown fox jumps over the lazy dog. Pack my box with five dozen liquor jugs.

EXPECT: Context menu includes "Copy" at the top because text is selected. Clicking "Copy" copies the selected text.
Test 11d: Context Menu Behavior

Additional checks:

  • Context menu items highlight on hover (different background color)
  • Clicking a menu item performs the action and closes the menu
  • Clicking outside the context menu closes it without performing any action
  • Only one context menu is visible at a time (right-clicking again replaces it)
EXPECT: Menu has hover highlighting, closes on click or click-away, and only one menu is visible at a time.
