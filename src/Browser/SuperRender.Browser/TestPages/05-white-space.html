Test 05: White-Space Modes

Back to index

Test 5a: white-space: normal (default)
Multiple spaces are collapsed to single spaces. Newlines are also collapsed. Everything wraps normally at the container edge. This is a long sentence to verify that wrapping occurs at the 400px boundary of the container.
EXPECT: All extra spaces and newlines are collapsed. Text wraps at the container edge.
Test 5b: white-space: pre
Spaces are preserved. Newlines are preserved too. Tab characters also preserved. Indented line. This will NOT wrap even if the line is very very very very very very very very very very very very long.
EXPECT: Spaces, tabs, and newlines are all preserved exactly as written. No wrapping even for long lines.
Test 5c: white-space: nowrap
This is a very long line of text that should not wrap at all regardless of how wide the container is. It should just keep going and going and extend beyond the box boundary. Spaces are collapsed but no line breaks.
EXPECT: All text on one line, no wrapping. Spaces collapsed to single spaces. Text may extend beyond the container.
Test 5d: white-space: pre-wrap
Spaces are preserved. Newlines are preserved too. But this mode DOES wrap when a line is too long for the container. This sentence should wrap at the 400px boundary while still preserving the spaces and newlines above.
EXPECT: Spaces and newlines are preserved, AND text wraps at the container edge.
Test 5e: white-space: pre-line
Spaces are collapsed like normal. But newlines ARE preserved. So each source line becomes a new line. Long lines will wrap at the container edge like white-space:normal does, this sentence should wrap at the 400px boundary.
EXPECT: Spaces collapsed to single space, but newlines are preserved. Wrapping occurs at container edge.
Test 5f: pre tag (UA stylesheet default: white-space: pre, monospace font)
function hello() {
    console.log("Hello, world!");
    for (var i = 0; i < 10; i++) {
        console.log(i);
    }
}
EXPECT: Code displayed in monospace font with all whitespace preserved (indentation, newlines).
