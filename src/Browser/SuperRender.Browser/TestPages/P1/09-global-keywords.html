Test 9a: color: initial (resets to default)

This text is red (inherited from parent).

This text uses color: initial (should be BLACK, the initial value).

This text is red again (inherited).

EXPECT: First and third paragraphs are red. The middle paragraph with color:initial is black (the CSS initial value for color).
Test 9b: width: inherit (copies parent value)
Parent: width 300px (blue border)
Child: width inherit (should also be 300px, red border)
EXPECT: The child box (red border) is the same width as the parent box (blue border), both 300px. The child explicitly inherits the parent width.
Test 9c: color: unset on inherited property
Parent has green text.

This paragraph has color: unset. Since color is an inherited property, unset = inherit, so this should be GREEN.

EXPECT: The paragraph inside the green parent is also green. "unset" on an inherited property (color) behaves like "inherit".
Test 9d: background-color: unset on non-inherited property
Parent has pink background.
This child has background-color: unset. Since background-color is NOT inherited, unset = initial, so background should be TRANSPARENT (showing parent pink through).
EXPECT: The child element has a transparent/initial background. Since it is inside the pink parent, the pink shows through. "unset" on a non-inherited property (background-color) behaves like "initial".
