Test 16a: Common entities
&amp; & ampersand
&lt; < less-than
&gt; > greater-than
&quot; " quotation mark
&apos; ' apostrophe
&nbsp; [ ] non-breaking space (between brackets)
EXPECT: Each entity renders as its correct character: & < > " ' and a non-breaking space.
Test 16b: Extended entities (typography)
&mdash; — em dash
&ndash; – en dash
&copy; © copyright
&reg; ® registered
&trade; ™ trademark
&hellip; … horizontal ellipsis
&laquo; « left guillemet
&raquo; » right guillemet
EXPECT: Each entity renders as its correct typographic character.
Test 16c: Mathematical entities
&sum; ∑ summation
&infin; ∞ infinity
&pi; π pi
&times; × multiplication
&divide; ÷ division
&plusmn; ± plus-minus
EXPECT: Mathematical symbols render correctly.
Test 16d: Greek letter entities
&Alpha; Α Alpha
&Beta; Β Beta
&Gamma; Γ Gamma
&Delta; Δ Delta
&Omega; Ω Omega
&alpha; α alpha (lowercase)
&beta; β beta (lowercase)
EXPECT: Greek capital and lowercase letters render correctly.
Test 16e: Arrow entities
&larr; ← left arrow
&rarr; → right arrow
&uarr; ↑ up arrow
&darr; ↓ down arrow
&harr; ↔ left-right arrow
EXPECT: Arrow characters render correctly in each direction.
