Test 09: Keyboard Shortcuts Reference

Back to index

Note: This page is a reference for the tester. The shortcuts are browser-level and not page-level, so they cannot be tested interactively from this page. Test each shortcut manually. On macOS, "Cmd" is the Command key. On Windows/Linux, use "Ctrl" instead.

Tab Management

Cmd+T Open a new tab
Cmd+W Close the current tab
Cmd+Tab Switch to the next tab
Cmd+Shift+Tab Switch to the previous tab

Navigation

Cmd+L Focus the address bar (select all text)
Cmd+R / F5 Reload the current page
Cmd+[ Go back in history
Cmd+] Go forward in history
Escape Unfocus address bar / dismiss

Content Scrolling

Up Arrow Scroll up by a small step
Down Arrow Scroll down by a small step
Page Up Scroll up by one page
Page Down Scroll down by one page
Space Scroll down by one page
Home Jump to the top of the page
End Jump to the bottom of the page

Developer Tools

F12 Toggle Developer Tools window
Cmd+Shift+I Toggle Developer Tools window (alternative)

Test Checklist

Verify each shortcut works by performing the action and observing the result:

  • Open 3 tabs with Cmd+T, switch between them with Cmd+Tab
  • Close a tab with Cmd+W
  • Focus address bar with Cmd+L, type a URL, press Enter
  • Navigate to a page, then use Cmd+R to reload
  • Navigate through links, use Cmd+[ and Cmd+] for back/forward
  • Load the scrolling test (07), test all scroll keys
  • Press F12 to open/close DevTools
