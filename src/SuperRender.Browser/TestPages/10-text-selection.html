Test 10: Text Selection

Back to index

How to test text selection:

  • Click and drag across text to select it
  • Selected text should appear with a blue highlight background
  • Right-click on selected text to see the "Copy" option in the context menu
  • Try selecting across paragraph boundaries
  • Try selecting text in different sections
Test 10a: Single paragraph selection

The quick brown fox jumps over the lazy dog. This is a classic pangram that contains every letter of the English alphabet. Try selecting any portion of this sentence by clicking and dragging your mouse from left to right.

EXPECT: Blue highlight appears behind the selected characters. Selection follows the mouse cursor position.
Test 10b: Multiple paragraphs

First paragraph: Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Second paragraph: Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

Third paragraph: Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur.

EXPECT: Selection can span across multiple paragraphs. Start in the first paragraph and drag down to the third.
Test 10c: Mixed formatting

Normal text, then bold text, then italic text, then underlined text, and back to normal.

Try selecting across the different formatting styles. The selection highlight should cover all text uniformly.

EXPECT: Selection highlight works consistently across bold, italic, underlined, and normal text.
Test 10d: Different font sizes

This is a heading (larger text)

This is normal body text. Try selecting from the heading down into this paragraph.

EXPECT: Selection works across different font sizes without visual glitches.
Test 10e: Long text for precision testing

ABCDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyz 0123456789

Try selecting exactly 5 characters. The selection should be character-accurate, not word-based.

EXPECT: Character-level precision when selecting. Each character boundary is individually selectable.
